# 分析
## 核心部件
1. CPU：对于办公场景（Office三件套、网页多开、视频会议），处理器的**单核性能**比多核性能更重要。不需要追求顶级的i9或R9，主流的 **Intel Core i5 / Ultra 5** 或者 **AMD Ryzen 7** 即可完美胜任，且发热量更低，噪音更小
2. 显卡：网页游戏和高清视频播放，**核显** 已经完全足够。不需要独显，加上独显反而会增加重量、发热和故障率。
3. 内存：建议16GB，现在的软件和网页占用内存越来越大，大内存能保证电脑用3-5年不卡顿。
4. 硬盘：至少512GB，推荐1TB，现在的系统和软件更新很快
## 外部接口
很多轻薄本为了薄，砍掉了USB-A口和HDMI口，必须具备 **USB-A口**（插老式U盘、鼠标接收器）、**HDMI口**（连接投影仪、电视）、**全功能Type-C**（支持充电）。如果能有 **RJ45网口**（插网线）则为加分项，适合办公环境。
## 售后与耐用
- 必须排除二线品牌（如机械革命、神舟等），虽然它们性价比极高，但小毛病多，售后网点少。
- 优选一线品牌（联想、惠普、荣耀/华为）的**商务本**或**主流全能本**系列。商务本通常经过军规测试（防摔、防潮），且售后政策往往比家用本更好（例如提供上门维修）。
